Описание

Кликните по блоку, состоящему минимум из двух кубиков одинакового цвета, чтобы они исчезли.
Стремитесь собрать блок как можно больше, так как за очередной кубик в блоке вы получаете больше очков,
чем за предыдущий.

Модули

game.py - Основная логика игры
gui.py - Графический интерфейс пользователя, реализованный на PyQt5
test.py - Unit-тесты, покрывающие игровую логику
